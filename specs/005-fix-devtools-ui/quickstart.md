# 快速开始: 验证修复

**功能分支**: `005-fix-devtools-ui`

## 验证步骤

1.  **启动示例**:
    ```bash
    npm run dev:browser
    ```

2.  **验证 Timeline**:
    *   打开 DevTools，切换到 Timeline 标签。
    *   确认能看到时间轴刻度和任务执行条。
    *   确认有一条垂直线（或滚动效果）表示时间流逝。

3.  **验证数据同步**:
    *   在控制台输入: `scheduler.stopTask('browser-cron')`
    *   观察 DevTools 列表中 `browser-cron` 的状态是否立即变为红色/停止图标。
    *   输入: `scheduler.startTask('browser-cron')`
    *   观察状态是否恢复。

4.  **验证 UI/Icons**:
    *   检查列表右侧的操作按钮。
    *   确认图标居中，无错位。
    *   确认 `Pause` 和 `Resume` 按钮根据状态正确切换。

5.  **验证主题**:
    *   点击太阳/月亮图标。
    *   确认背景变黑（深色模式），文字变白。
    *   确认任务 ID 字体为等宽字体。

6.  **验证停靠**:
    *   点击头部新增加的“停靠切换”按钮（图标可能为 `[◳]` 或类似）。
    *   确认面板移动到底部。
    *   再次点击，确认回到右侧。
